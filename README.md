# Retro Games

Statyczna strona w stylu pixel-art z katalogiem klasycznych gier, filtrami, galerią lightbox i efektem CRT.

## Uruchomienie lokalne

1. Sklonuj repozytorium.
2. Otwórz plik `index.html` w przeglądarce **lub** uruchom prosty serwer:

```bash
python3 -m http.server 8000
```

Następnie wejdź na `http://localhost:8000`.

## Publikacja na GitHub Pages

1. Wypchnij repo na GitHub.
2. Wejdź w **Settings → Pages**.
3. Wybierz źródło: **Deploy from a branch**.
4. Ustaw branch `main` (lub aktualny) i folder `/root`.
5. Zapisz — po chwili strona będzie dostępna pod adresem GitHub Pages.

Pliki projektu:
- `index.html`
- `style.css`
- `script.js`
